## **üõ†Ô∏è Technical Highlights**

### **1\. Advanced API Integrations**

These projects leverage public APIs to provide real-world functionality without backend overhead:

* **Jikan (MyAnimeList):** Powers AniStream Hub for real-time anime data.  
* **Gutendex (Project Gutenberg):** Drives the LibriVerse e-book library.  
* **RandomUser.me:** Generates realistic user profiles for VibeCircle and VelvetRooms.  
* **CoinGecko:** Provides live crypto pricing for CryptoFlow.  
* **ESPN (Hidden API):** Fetches live sports scores for StreamArena.

### **2\. Authentication Simulations**

Many apps feature a realistic **OAuth** flow (Google/Guest) that persists user sessions via localStorage. This allows for features like:

* Persistent Watchlists/Favorites.  
* User Profiles (Avatar, Name, Email).  
* Unlocked "Premium" features.

### **3\. Responsive & Adaptive UI**

Built with **Tailwind CSS**, every application is mobile-first:

* **Dark/Light Mode:** System-aware theme toggles.  
* **Mobile Navigation:** Bottom bars for mobile, sidebars for desktop.  
* **Snap Scrolling:** Horizontal scrolling for media cards.

## **üöÄ Installation & Usage**

To run any of these projects locally:

1. **Clone the repository** containing the project files.  
2. **Open index.html** in any modern browser (Chrome, Firefox, Edge).  
3. *Note:* No Node.js or backend server is required. These are "Serverless Frontend" applications.

\# Example: Running via Python SimpleHTTPServer  
cd path/to/project  
python3 \-m http.server 8000  
\# Open localhost:8000 in your browser
